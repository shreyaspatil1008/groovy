#Installation- JDK8 must be pre-installed and update JAVA_HOME
Download 
https://gradle.org/next-steps/?version=4.8&format=bin  => Unzip to c:\Gradle 
https://dl.bintray.com/groovy/maven/apache-groovy-binary-2.5.0.zip  => Unzip to c:\Groovy 

Update System Path env variable to have below string 
C:\Gradle\gradle-4.8\bin\;C:\Groovy\groovy-2.5.0\bin\
Note after updating Path, open a cmd prompt and type 
$  gradle --version 
$ groovy --version 
If above do not work, there is some problem either in PATH setting or PATH value is wrong 
Find correct location of groovy.exe and groovyConsole.exe from C:\Groovy and C:\Gradle and update the PATH correctly 
Also install notepad++.exe and/or any IDE as demanded by trg participants
